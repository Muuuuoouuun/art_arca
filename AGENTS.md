# Project: art_hub (Next.js)

## 🏢 Project Context
This is a Next.js project called `art_hub`. It uses Tailwind CSS for styling and TypeScript for development.

## 🛠 Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS / PostCSS
- **Language:** TypeScript
- **Package Manager:** npm (package-lock.json)

## 📁 Key Folders
- `src/`: Application source code
- `public/`: Static assets
- `scripts/`: Project scripts

## 🏗 Coding Standards
- Use functional components and hooks for React.
- Prefer Tailwind utility classes for styling.
- Ensure TypeScript types are correctly defined.
- Follow a consistent file structure under `src/`.

- **Dev:** `npm run dev`
